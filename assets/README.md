# Brand assets

Public-facing JankMark brand art. The mark is a green frame-rate waveform broken
by one tall red "jank" spike.

| File | What | Use |
|---|---|---|
| `logo_lockup.png` | Horizontal lockup: mark + wordmark + tagline (dark bg) | README header, docs |
| `wordmark_dark.png` | Wordmark only — "Jank" light, "Mark" red (dark bg) | Headers on dark surfaces |
| `banner_social.png` | Wide social / repo banner (dark bg) | GitHub **Settings → Social preview** |
| `icon.png` | 512px app icon | Favicon, docs |
| `icon_256.png` | 256px app icon | Smaller raster use |
| `logo_icon.svg` | Vector sketch of the mark | Reference only |

> These PNGs have **solid dark backgrounds** (intentional — they sit on dark
> surfaces). Don't strip the backgrounds.
