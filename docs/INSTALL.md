# Installing JankMark

1. Download the latest installer from the
   [Releases](https://github.com/AbiMangalan/JankMark/releases/latest) page.
2. Run `JankMark_Setup_<version>.exe`.
3. Follow the installer prompts.
4. Launch JankMark from the Start Menu or desktop shortcut.

## "Windows protected your PC" / unknown publisher
Until the app is code-signed, Windows SmartScreen may warn you because the
publisher isn't yet recognized. If you trust this download:
1. Click **More info**.
2. Click **Run anyway**.

Code signing is planned so this warning goes away in a future release.

## Uninstalling
Use **Settings → Apps → Installed apps → JankMark → Uninstall**, or the
"Uninstall JankMark" Start Menu entry.

Your settings are stored in `%USERPROFILE%\.jankmark\` and can be deleted
manually if you want a fully clean removal.
